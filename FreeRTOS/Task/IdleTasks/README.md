# IdleTasks Example

Calls the `vApplicationIdleHook() API` whenever the Tasks are in blocked state

### Functions Used

`vApplicationIdleHook() API`

### Setting up

1. `make menuconfig`

2. `Component config`

3. `FreeRTOS`

4. `Use FreeRTOS legacy hooks` --> Press y/Y

5. `Enable legacy idle hook` --> Press y/Y
